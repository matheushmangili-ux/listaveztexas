Três direções para o novo dashboard

Protótipos standalone para avaliar antes de implementar na app. Cada um usa os mesmos tokens (paleta roxa dark/light) e os mesmos dados falsos, mas propõe uma estética diferente. Alterne dark/light no canto para ver ambos.

01 · analítico

Tremor-like

Paleta qualitativa (6 cores), sparkline em cada KPI card, meta line no ring de conversão, cards com delta pelo sinal. Inspirado em tremor.so + Stripe Dashboard.

02 · minimal

Linear-like

Minimalismo radical, tipografia forte, monocromia + 1 accent, hero KPI grande, grid denso, sem greeting casual. Inspirado em linear.app/analytics.

03 · pro híbrido

Stripe-like

Sparkline preenchido grande em cada KPI, comparação temporal lado a lado (hoje vs ontem), multi-cor pastel, annotations prominentes. Tom premium/financeiro.

Observações: (a) dados nos mockups são fictícios; (b) cada mockup usa o tokens.css do app — as cores roxo/accent vão mudar automaticamente se trocar os tokens; (c) depois de escolher uma direção, custo de implementação na app real: 6-15h dependendo do escopo final.
